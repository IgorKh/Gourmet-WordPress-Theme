Фреймворк для back end'а. Строит настройки и обеспечивает АЯКС запросы.

== Description ==

Сделан для удобства построения бэк енда.

Вся суть состоит в массиве настроек, который передается конструктору.


Каждый блок может содержать:
* title - заголовок
* fields - другие блоки, которые будут вложены в этот
* type - если поле конечное (т.е не содержит fields), то указываем тип поля.
 - input - однострочное текстовое поле. указывается по умолчанию
 - checkbox: чекбокс
 - image: загрузка изображения
 - number: поле с номером
 - textarea: текстовое поле
 - wpPages_dropDown: выпадающий список страниц сайта

* size - размер блока (в размерах модульной сетки для bootstrap) от 1(самая маленькая еденица) до 12 (самая большая еденица)
* count или max - может указываться одно из двух. Если указан count - количество блоков, будет равно указанному.
Если указан max, то блоки могут добавляться или удаляться, с любым уровнем вложения

* title_add если указан max, то сожержит текст кнопки добавления блока
* title_remove если указан max, то сожержит текст кнопки удаления блока

Имеет вид:
Первым элементом идет заголовк всей страницы настроек
Потом идут большие секции настроек, они не могут иметь значения, кроме title и fields
Далее уже идут универсальные блоки

Главнный массив настроек
static $parts = array(
    'title'=>'',
    'menu' => array(
        'title' => 'А это уже блок настроек, который может содержать еще блоки без лимита',
        'fields' => array(
            'logo' => array(
                'title' => 'логотип',
                'size' => 10,
                'type' => 'image'
            ),
        ),
    ),
);
